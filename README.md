# Netflix Dashboard – Tableau Project

## Tools Used
- Tableau
- Excel

## Project Description
This interactive dashboard analyzes Netflix Movies and TV Shows based on:
- Country
- Genre
- Ratings
- Release Year
- Type (Movie / TV Show)

## Key Insights
- Movies dominate Netflix content (~68%)
- Most content was added after 2016
- Documentaries and Stand-Up Comedy are top genres
- USA and India contribute the highest content
